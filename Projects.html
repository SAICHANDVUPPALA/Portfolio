Description

In this project I along with a team of 6 members have developed a Mobile Application "FarmersConnect" using the React Native. The mobile application is built on an idea to bring the farmers at one platform and be able to share their thoughts, view, experiences on farming which might be useful to other farmers. This application also shares data regarding the NCDEX prices updates, Banned Products, Crop Diseases and other important information for a Farmer. I have been part of designing the UI screens for the project and also developed the SignUP Page.

We have used Figma and BuilderX to create the UI Screens for the Application.

In this project we have used Firebase to authenticate the Users either by email and password, Google SignIn, SMS Verification. Firebase was also helpful for us to store the data in the Firestore Database and store files in the Storage. I have Developed the SMS Authentication and implemented the Recaptcha Verifier while logging in using the SMS Verification where captcha (Mathematical question) was used with normal signup.

I have developed post service as a User (Farmer) can share his views through the posts and every other user can view an comment on the Post. A user can upload an Image or a Video with the Post. I have stored the Images and Videos in the Storage and Post related information in the Firestore Database. Whenever Feed is opened, the data is retrieved from the Firestore and sent to a flatlist which aligns the data in the required form to a visual experience.
